# Concepts and Terminology

## Von Neumann Computer Architecture
Both program instructions and data are kept in electronic memory. 

Comprised of 4 main components: 
1) Memory
2) Control Unit
3) Arithmetic Logical Unit ( 2 and 3 make up cpu )
4) Input and Output

Parallel computers still follow this basic design, just multiplied in units. The basic, fundamental architecture remains the same.
